# Cybersecurity Basics

## CIA Triad
| Principle | Definition | Real-world Violation |
|-----------|-----------|---------------------|
| Confidentiality | Data is accessible only to authorized users | 2017 Equifax breach — 147M SSNs exposed |
| Integrity | Data is accurate and unaltered | Stuxnet malware modified PLC code in Iranian nuclear centrifuges |
| Availability | Systems and data are accessible when needed | 2016 Dyn DDoS — took down Twitter, Netflix, GitHub |

## Threat Types
| Threat | How it works | Real incident |
|--------|-------------|---------------|
| Phishing | Spoofed email tricks user into revealing credentials | 2016 DNC hack via spear-phishing email |
| Malware | Malicious software installed on a victim system | WannaCry ransomware — 200K systems in 150 countries |
| DDoS | Floods a target with traffic until it crashes | Mirai botnet attack on Dyn DNS, Oct 2016 |
| SQL Injection | Malicious SQL inserted into input fields to query DB | 2008 Heartland Payment Systems — 130M card numbers stolen |
| Brute Force | Systematically tries all possible passwords | 2012 LinkedIn breach — 6.5M hashed passwords cracked |
| Ransomware | Encrypts victim files, demands payment for key | Colonial Pipeline 2021 — $4.4M ransom paid |

## Attack Vectors
| Vector | Description |
|--------|-------------|
| Social Engineering | Manipulates humans rather than systems — pretexting, baiting, tailgating |
| Wireless Attacks | Exploits Wi-Fi weaknesses — evil twin APs, WPA2 handshake capture, deauth attacks |
| Insider Threats | Malicious or negligent actions by authorized users — data theft, sabotage |
